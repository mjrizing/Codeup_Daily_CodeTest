# 코딩테스트란?
- 주어진 시간동안 요구사항에 맞게 프로그래밍하여 Accept나 점수를 받는 시험!

# 필요한역량과 절차
- 추상화(상황분석)
- 절차적 사고
- 구현

# 대표 유형
- 파싱, 해싱, 정렬, 시뮬레이션 : 구현
- 탐색(BFS/DFS), 완전탐색(백트래킹) : 탐색
- 자료구조(스택,큐,힙 등) : 구조
- Greedy, DP, 이분탐색 : 알고리즘

# 분명 기초적인 개념인데 왜 못풀까?

- 코딩테스트도 국어시험이라 생각해보자!

비교적 어려운 문법, 언어 자체속도는 빠름, STL 등의 장점

# Python

# Java
